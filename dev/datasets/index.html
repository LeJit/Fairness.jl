Fairness Datasets

To make it easy to try algorithms and metrics on various datasets, MLJFair shall be providing with the popular fairness datasets.

These datasets can be easily accesses using macros.

Following datasets shall be available in future through macros.

  • COMPAS Dataset
  • Adult Dataset
  • German Dataset
  • Bank Dataset

But currently, only a toy dataset is available.

Toy Data

This is a 10 row dataset that was used by authors of Reweighing Algorithm.

MLJFair.@load_toydata — Macro

Macro to read csv file of job data and convert columns to categorical. Returns the tuple (X, y, ŷ)

source
MLJFair.@load_toyfairtensor — Macro

Macro to create fairness Tensor for jobs.csv The fairness tensor will be created on the basis of the column Job Type. This column has 3 different values for job types.

source
